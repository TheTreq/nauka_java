# INTELLIJ IDEA — skróty klawiszowe

## `Shift + F6` — zmiana nazwy (Rename)
```
Shift + F6
```
- Zmienia nazwę zaznaczonego elementu kodu — klasy, metody, zmiennej i innych.
- Działa inteligentnie: zmienia nazwę wszędzie, gdzie dany element jest używany w projekcie (nie trzeba ręcznie poprawiać każdego miejsca osobno).
